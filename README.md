# TROVYN Homepage

Welcome to the TROVYN homepage project! This project is a modern, responsive website for a creative tech studio named TROVYN, built using HTML5, Tailwind CSS, and vanilla JavaScript.

## Project Structure

The project is organized as follows:

```
trovyn-homepage
├── public
│   └── index.html          # Main HTML document
├── src
│   ├── assets
│   │   └── placeholder.jpg  # Placeholder image for sections
│   ├── components
│   │   ├── Navbar.js       # Sticky navigation bar component
│   │   ├── Hero.js         # HERO section component
│   │   ├── About.js        # About section component
│   │   ├── Contact.js      # Contact section component
│   │   └── Footer.js       # Footer component
│   ├── styles
│   │   └── main.css        # Main CSS styles including Tailwind CSS
│   ├── app.js              # Main JavaScript entry point
│   └── layout.js           # Layout component wrapping main content
├── tailwind.config.js       # Tailwind CSS configuration
├── package.json             # npm configuration file
└── README.md                # Project documentation
```

## Features

- **Responsive Design**: The website is designed to be mobile-first and responsive, ensuring a great user experience on all devices.
- **Dark Theme**: The project features a premium dark theme design, enhancing the visual appeal.
- **Smooth Scrolling**: The sticky navbar allows for smooth scrolling to each section of the homepage.
- **Placeholder Content**: The sections include placeholder text and images, which can be easily replaced with actual content.

## Setup Instructions

1. **Clone the Repository**: 
   ```bash
   git clone <repository-url>
   cd trovyn-homepage
   ```

2. **Install Dependencies**: 
   ```bash
   npm install
   ```

3. **Run the Development Server**: 
   ```bash
   npm start
   ```

4. **Open in Browser**: Navigate to `http://localhost:3000` to view the project.

## Customization

- You can customize the color palette and typography by modifying the `tailwind.config.js` file.
- Update the placeholder images and text in the respective components to reflect the actual content of TROVYN.
